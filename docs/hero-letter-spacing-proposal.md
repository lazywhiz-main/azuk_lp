# ヒーローセクション 字間（letter-spacing）調整提案

## 📊 現状分析

### 現在の実装
- **値**: `tracking-[0.125em]` = **0.125em**（約2px相当、フォントサイズ16px基準）
- **場所**: `HeroSection.tsx`の`h1`要素

### 参考値（元のHTML）
- **`lp.html`**: `letter-spacing: 2px`（固定値）
- **`lp_updated_20260103.html`**: `letter-spacing: 0.08em`（約1.28px相当）

## 💡 調整案

### オプション1: 新しいLPに合わせる（推奨）
- **変更後**: `tracking-[0.08em]`
- **縮小幅**: 0.125em → 0.08em（**36%縮小**）
- **理由**: 新しいLPのデザインに合わせる

### オプション2: 控えめに縮める
- **変更後**: `tracking-[0.1em]`
- **縮小幅**: 0.125em → 0.1em（**20%縮小**）
- **理由**: 少しだけ縮めて、より読みやすくする

### オプション3: Tailwind標準クラスを使用
- **変更後**: `tracking-wide`（0.025em）
- **縮小幅**: 0.125em → 0.025em（**80%縮小**）
- **理由**: 標準クラスを使用し、大幅に縮める

### オプション4: さらに控えめに
- **変更後**: `tracking-[0.09em]`
- **縮小幅**: 0.125em → 0.09em（**28%縮小**）
- **理由**: オプション1と2の中間

## 📏 視覚的な比較

| オプション | 値 | 縮小率 | 視覚的な印象 |
|-----------|-----|--------|------------|
| **現状** | 0.125em | - | やや広め |
| **オプション1** | 0.08em | 36%縮小 | 新しいLPと同じ |
| **オプション2** | 0.1em | 20%縮小 | 控えめに縮小 |
| **オプション3** | 0.025em | 80%縮小 | 大幅に縮小 |
| **オプション4** | 0.09em | 28%縮小 | 中間 |

## 🎯 推奨

**オプション1（`tracking-[0.08em]`）を推奨**

### 理由
1. 新しいLP（`lp_updated_20260103.html`）のデザインに一致
2. 36%の縮小で、視覚的に適度な調整
3. 読みやすさとデザイン性のバランスが良い

### 実装例
```tsx
// 変更前
<h1 className="... tracking-[0.125em] ...">

// 変更後
<h1 className="... tracking-[0.08em] ...">
```

## 🔍 注意点

- サブテキスト（`p`要素）の字間は変更しない（現状のまま）
- モバイルとPCで同じ値を使用（レスポンシブ対応は不要）
